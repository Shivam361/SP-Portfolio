Context

Overview & goals

The primary goal was to build a digital presence that accurately reflects my background in Software Engineering, Game Development, and UI/UX. Rather than using generic templates or heavy frameworks, I chose to engineer the site from the ground up using core web technologies.

This approach ensures lightning-fast load times, a highly customized "dark gaming" aesthetic, and complete control over accessibility features, memory leaks, and render-blocking resources.

Engineering

Performance & Accessibility

Great design shouldn't come at the cost of usability. This portfolio implements modern accessibility features specifically targeting contrasting needs, reading fatigue, and vestibular disorders.

Feature Technical Implementation
Vestibular Safety window.matchMedia('(prefers-reduced-motion: reduce)') actively disables 3D tilts and intensive animations.
WCAG Contrast Body text colors optimized against `#08080d` backgrounds to maintain \>4.5:1 ratios for prolonged readability.
Battery Optimization Pointer-detection (`media(pointer: fine)`) automatically breaks `requestAnimationFrame` simulation loops on mobile devices.
Keyboard Navigation Generous :focus-visible ring offsets ensure smooth, unobstructed 'Tab' navigation for screen readers.
