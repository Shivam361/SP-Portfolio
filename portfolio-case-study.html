Projects / Case study

Portfolio Development

A high-performance, cinematic Single Page Application (SPA) portfolio website built with Swup.js, GSAP, and Three.js. Designed with a unified neon aesthetic, rigorous WCAG-compliant accessibility standards, and optimized for 60FPS performance.

Solo HTML / CSS / JS UI / UX Accessibility & Web Perf

Context

Overview & goals

The primary goal was to build a digital presence that accurately reflects my background in Software Engineering, Game Development, and UI/UX. By leveraging modern tools like Swup.js and GSAP, the site achieves seamless transitions, cinematic animations, and a highly interactive user experience.

This approach ensures lightning-fast load times, a highly customized "dark gaming" aesthetic, and complete control over accessibility features, memory management, and render-blocking resources.

Capabilities

Technical Highlights

SPA Architecture

  • Swup.js Seamless Routing for cinematic page transitions with zero layout shift.
  • Smart Hash Routing enabling intelligent navigation between deep-linked sections and distinct pages.
  • UI Singleton Pattern caching persistent DOM elements to eliminate redundant queries.
  • Lifecycle Destructors ensuring WebGL backgrounds and event listeners are safely destroyed and recreated to prevent memory leaks.

Performance & Interactions

  • Consolidated IntersectionObservers driving scroll-spy navigation and staggered reveals without performance overhead.
  • Vanilla JS HTML5 Canvas particle networks maintaining consistent 60FPS.
  • Custom magnetic cursor with dynamic pointer-detection to enhance desktop UX while preserving mobile battery life.
  • Render-blocking libraries (like AOS) were entirely purged in favor of native GSAP and CSS-accelerated transforms.

Engineering

Performance & Accessibility

Great design shouldn't come at the cost of usability. This portfolio implements modern accessibility features specifically targeting contrasting needs, reading fatigue, and vestibular disorders.

Feature Technical Implementation
Vestibular Safety window.matchMedia('(prefers-reduced-motion: reduce)') actively disables intensive CSS/JS animations and parallax effects.
WCAG Contrast Body text colors optimized against `#08080d` backgrounds to maintain >4.5:1 ratios for prolonged readability.
Battery Optimization Pointer-detection (`media(pointer: fine)`) automatically breaks `requestAnimationFrame` simulation loops on mobile devices.
Keyboard Navigation Generous :focus-visible ring offsets ensure smooth, unobstructed 'Tab' navigation for screen readers.

Stack

Technologies & Artifacts

Build & Delivery

  • Swup.js for true Single Page Application (SPA) routing and cinematic view transitions.
  • Lenis & GSAP providing the foundation for custom tick loops and fluid scroll synchronization.
  • Three.js & Vanta.js orchestrating the interactive webGL particle environments.
  • SplitType & Typed.js for robust, accessible text manipulation and typewriting engines.
  • Modular CSS3 for highly targeted, component-scoped styling and Custom Properties tracking.
  • Zero CDN Runtime dependencies — all assets are bundled and served locally for ultra-fast Time-to-Interactive (TTI).
