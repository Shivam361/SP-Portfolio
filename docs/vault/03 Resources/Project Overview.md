# Project Overview: SP-Portfolio 🏛️

## Summary
A high-performance, cinematic **Single Page Application (SPA)**, dark-themed personal site for Shivam Parab.

## 🛠️ Tech Stack
- **Architecture**: Swup.js (SPA transitions)
- **Animation**: GSAP & ScrollTrigger
- **Scrolling**: Lenis Smooth Scroll
- **WebGL**: Three.js + Vanta.js (Particle backgrounds)
- **Styling**: Vanilla CSS3 (Modular Architecture)
- **Typography**: Google Fonts (Inter, etc.)

## 📂 Repository Structure
- `index.html`: Main portfolio interface.
- `css/`: Modular stylesheets (base, components, vendor).
- `js/`: Core interaction engine and vendor dependencies.
- `img/`: Optimized visual assets.

## 🏗️ Technical Architecture
- **Swup.js Core**: True SPA experience with seamless content swapping.
- **GSAP & ScrollTrigger**: Hardware-accelerated reveals.
- **Lenis Smooth Scroll**: Persistent momentum across navigation.
- **Smart Lifecycle Hooks**: Manual destruction of WebGL contexts to prevent memory leaks.
- **Zero CDN Dependency**: All assets shipped locally in `/vendor`.
