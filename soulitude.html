The Challenge

Overview & Scope

Conceived and pushed to release during the intensive Staffordshire University CGD22 Game Jam, Soulitude tests a player's endurance as they navigate procedurally generated, rogue-like 3D dungeons. Game jams inherently challenge developers to optimize architecture immediately, preventing bloated codebases while collaborating in parallel across multi-disciplinary teams.

My Role: Senior Programmer

Operating as the Senior Programmer among a collaborative group, I took ownership of the entire UI Architecture and Quality Assurance pipelines:

  • Minimap Systems: Programmed real-time orthographic minimap matrices that bind actively to player positional data against the dungeon layout, providing vital navigation info without memory hemorrhage.
  • HUD & Widget Blueprints: Engineered all User Interfaces (HUDs, Menus, health/stamina tracking bars) using Unreal Motion Graphics (UMG), ensuring multi-resolution responsiveness.
  • QA & Iteration: Instituted rigorous, high-frequency gameplay logic integration tests to pinpoint clipping errors, logic overlaps, and UI desynchronization amidst team commits.

Core Technology Stack

  • Unreal Engine: Core engine runtime, Blueprint visual scripting logic.
  • UMG (Unreal Motion Graphics): State-bound widget components and dynamic anchors.
  • Version Control (Git/Perforce): Multi-developer repository branching.
  • Agile Collaboration: Rapid-sprint iteration loops during the strict 48-hour jam constraints.
